# Deporty UI (Solo pantallas visuales)

Este proyecto fue simplificado para conservar solo las pantallas visuales (HTML/Svelte + Tailwind). No hay backend, base de datos ni APIs.

## Pantallas disponibles

- Home (landing visual)
- Login (interfaz visual)

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run check
npm run lint
npm run format
```
